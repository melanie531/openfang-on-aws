OpenFang is an open-source Agent Operating System built in Rust that compiles to a single ~32MB binary. Unlike traditional chatbot frameworks, it runs autonomous agents called "Hands" — pre-built capability packages that operate on schedules without user prompting, performing tasks like deep research, lead generation, OSINT collection, and social media management. The system features 14 crates, 137K lines of code, 16 security layers including a WASM sandbox, 40 channel adapters, 27 LLM provider integrations, and a built-in dashboard. It can be deployed on AWS EC2 with Bedrock via a LiteLLM proxy sidecar.
